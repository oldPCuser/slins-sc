INSTALEM LINUX NOS COMPUTADORES

Olá, sou um aluno da escola Papa João Paulo II, que percebeu que os computadores da sala de informática não usam uma distribuição Linux Educacional. Alguns alunos têm difilculdade de usar um computador desktop, e o sistema operacional Windows é péssimo em Acessibilidade.

As distribuições Linux (Maioria baseados no Debian) são Gratuitos e de Código Aberto, assim, podem ser aproveitados ao máximo com ferramentas poderosas e úteis, como Gimp, LibreOffice, VLC, Kodi, Nautilus (GNOME)... Bom, se citar todos os programas ficarei até amanhã escrevendo. Os sistemas com WMs baseados em GTK2 e GTK3 são amplamente customizáveis.

Por isso decidi fazer esse pacote de ferramentas para mostrar a vocês, alternativas capazes de substituir o Windows em todos os quesitos.

Abaixo terá uma lista completa de recomendações de distribuições 100% gratuitas e de código aberto. ótimas para uso educacional:

ENDLESS OS

Sistema Especial para quem NUNCA mexeu em um computador. Baseado com pesquisas em todo o mundo

Características

Navegador Interface Office Tamanho da ISO
Chromium GNOME Shell LibreOffice 13GB

ubermix (em inglês)

Sistema especial para crianças aprenderem a mexer no desktop. Com programas e jogos educativos com tutoriais já acompanhados no sistema

Características

Navegador Interface Office Tamanho da ISO
Firefox GNOME Shell LibreOffice 7GB
